In case an error occured PSX will always return the following response format. If the "psx_debug" parameters is false the trace and context parameter are not available also the message is an general error message if the exception is not an DisplayException.

{
    "success": false,
    "title": "PSX\\Http\\Exception\\NotFoundException",
    "message": "Post does not exist in [...]\\src\\WordpressGateway\\Service\\PostManager.php on line 97",
    "trace": "[...]",
    "context": "[...]"
}
